# fastify-demo-app
Простое приложение на Fastify с кнопкой для API запроса.

## 📋 Описание

Приложение состоит из сервера на Fastify и клиентской части. 
При нажатии на кнопку отправляется запрос на сервер, и при успешном ответе:
- В консоли браузера появляется сообщение "Запрос прошел успешно"
- Под кнопкой появляется надпись "Запрос прошел успешно ✅"

Для запуска сервера - `npm start`
